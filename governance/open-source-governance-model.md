# Frasberg AI Open Source Governance Model

## 1. Governance Structure

- Governance Council — final authority
- Luchii Directorate — model architecture
- Infrastructure Authority — deployment & CI/CD
- Domain Stewards — Species, Civilizations, Chronicles

## 2. Contribution Requirements

- Signed commits
- PR review
- Governance checklist
- Safety validation
- Code quality severity: Errors only
- No direct pushes to protected branches

## 3. Transparency

Public repos include:

- API documentation
- SDKs
- Portal UI
- Public governance summaries
- Release notes

Private repos include:

- Constitutions
- Safety systems
- Legal frameworks
- Internal codex architecture

## 4. Licensing

- Enterprise License for commercial use
- Open-source license for public tools
- Trademark protection for Frasberg AI brand

## 5. Safety

- Redaction engine
- Policy engine
- Multi-region compliance
- Audit logs
- Governance enforcement

## 6. Community Interaction

- Public issues
- Public discussions
- Contribution guidelines
- Code of conduct
- Governance transparency reports
